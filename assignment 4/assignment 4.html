CSS Box Model

This page demonstrates the CSS box model, showing the effects of padding, border, margin, and the difference between `content-box` and `border-box` models.

Content-Box Model (Default)

This box uses the content-box model. The width is set to 300px, and the padding, border, and margin are applied outside of the content width.

  • Width: 300px
  • Padding: 20px
  • Border: 5px solid blue
  • Margin: 20px

Border-Box Model

This box uses the border-box model. The width includes the padding and border, so the total width of the box remains 300px.

  • Width: 300px (including padding and border)
  • Padding: 20px
  • Border: 5px solid blue
  • Margin: 20px
